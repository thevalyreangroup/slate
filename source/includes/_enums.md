# Enums

The HeartZones API uses the following enums:

## Type
`type`

Used for assigning a `type` to a user account.

Value | Name
----- | ----
0 | Admin
1 | Instructor
2 | Participant

## Access Level
`accessLevel`

The account's level of access to specified API endpoints

Value | Name
----- | ----
0 | Admin
1 | Support Admin
2 | Support
3 | Supervisory Admin
4 | Instructor Admin
5 | Instructor
6 | Participant
7 | Disabled

## CLASS TYPE
`type`

Value | Name
----- | ----
0 | Scheduled
1 | Impromptu
